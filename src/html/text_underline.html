  • 1. linear-gradient 속성을 이용해서 밑줄을 긋는다.
  • 2. 내부 a.example이 inline인 경우 줄바꿈(br 가능) 되고도 밑줄을 긋는다.(4, 5번째)
  • 3. 텍스트 영역에 hover해서 밑줄 확인 가능하다.
  • 4. 밑줄의 위치를 조절할 수 있는지 확인이 필요하다.
  • 4-1. 이 부분이 해결되어야 활용도가 높다.
  • 4-2. 텍스트 영역의 크기가 커져야 될 것 같다.(예상)
